# ToDoアプリ 仕様書

## 1. プロジェクト概要

### 1.1 プロジェクト名
ToDoApp - PWA対応ローカルToDoリストアプリ

### 1.2 目的
当日または翌日以降のタスクをカレンダー形式で管理し、直感的な操作でタスク完了を管理できるPWAアプリケーション

---

## 2. 技術仕様

### 2.1 使用技術
| カテゴリ | 技術選定 |
|----------|----------|
| **フレームワーク** | Next.js 14+ (App Router) |
| **UIライブラリ** | shadcn/ui |
| **CSS** | Tailwind CSS |
| **状態管理** | Zustand (ローカル状態) |
| **データ保存** | IndexedDB (Dexie.js) |
| **カレンダー** | react-day-picker または date-fns |
| **PWA** | next-pwa |
| **デプロイ** | Cloudflare Pages (GitHub連携) |

### 2.2 アーキテクチャ
- **SPA構成**: クライアントサイド完結
- **データ永続化**: IndexedDBを使用したローカルストレージ
- **オフライン対応**: Service Workerによるキャッシュ
- **レスポンシブ**: スマートフォン優先設計

---

## 3. 機能要件

### 3.1 カレンダー表示
- 当月のカレンダーを表示
- 過去の日付はグレーアウト（または非表示）
- タスクがある日付にインジケーター表示

### 3.2 タスク登録
- 日付選択からタスク追加画面へ遷移
- タスク名入力（必須、最大100文字）
- 日付設定（デフォルト: 選択日）
- 時刻設定なし（イベントタイム方式）
- タスク追加時にリストに即時反映

### 3.3 タスク表示
- 選択日のタスク一覧を表示
- 未完了タスクと完了済みタスクを分けて表示
- 完了済みタスクはグレーアウト表示

### 3.4 タスク操作
- **完了ボタン**: タップで完了済みリストへ移動
- **未完了に戻す**: 完了済みタスクのタップで未完了へ戻す
- **削除**: タスク削除機能

### 3.5 ナビゲーション
- 月表示切り替え（前月/翌月）
- 今日へボタン
- 日付選択でその日のタスク一覧を表示

### 3.6 PWA機能
- ホーム画面への追加可能
- スタンドアローン表示
- オフライン動作

## 4. データ構造

### 4.1 Task インターフェース
```typescript
interface Task {
  id: string;           // UUID
  title: string;        // タスク名
  date: string;         // YYYY-MM-DD形式
  completed: boolean;   // 完了フラグ
  createdAt: number;    // 作成タイムスタンプ
  updatedAt: number;    // 更新タイムスタンプ
}
```

### 4.2 IndexedDB スキーマ
```typescript
const dbSchema = {
  name: 'todoapp-db',
  version: 1,
  stores: {
    tasks: 'id, date, completed, createdAt'
  }
};
```

## 5. UI/UX仕様

### 5.1 画面構成
| 画面 | 説明 |
|------|------|
| **メイン画面** | カレンダー + 選択日のタスク一覧 |
| **タスク追加** | タスク名入力 + 日付選択モーダル |

### 5.2 カラーパレット
```css
--primary: #3B82F6;      /* 青 */
--primary-dark: #2563EB;
--success: #10B981;      /* 緑（完了） */
--gray-100: #F3F4F6;
--gray-500: #6B7280;
--gray-900: #111827;
```

### 5.3 操作フロー
```
1. アプリ起動 → 当日のカレンダー表示
2. 日付タップ → その日のタスク一覧を表示
3. 「+」ボタンタップ → タスク追加モーダル表示
4. タスク名入力 → 追加ボタンタップ
5. タスク一覧に即時反映
6. タスクのチェックボックス/完了ボタンタップ → 完了済みへ移動
```

### 5.4 タッチ操作
- ボタンタッチエリア: 最小44px
- スワイプでの月切り替え（オプション）
- ロングプレスでタスク削除（オプション）
